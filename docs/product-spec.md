# Product Specification

## 1. システム概要

社内勤怠入力およびfreee人事労務への連携を行うWebアプリケーション。

---

## 2. 認証

- LINE WORKS SSO を利用
- freee連携は個人設定画面で登録
- 未設定の場合はfreee連携不可

---

## 3. 権限

| ロール   | 権限                       |
|----------|----------------------------|
| employee | 自分の勤怠入力・修正       |
| manager  | 自部署メンバーの閲覧・修正 |
| admin    | 全社員閲覧・修正・締め解除 |

部署情報および役職情報はアプリ内ユーザーテーブルで管理する。

---

## 4. 勤怠入力

### 入力形式
- 月次一覧形式
- 日次詳細入力可能

### 入力補助
- 前日コピー
- 前月コピー
- 勤務パターン
- 祝日自動反映

### バリデーション
- 出勤 < 退勤
- 勤務日のみ必須
- 未入力日は強調表示

---

## 5. 締め処理

- 社員別 × 月単位
- manager / admin が締め可能
- admin のみ解除可能
- 締め後はemployee編集不可

---

## 6. freee連携

### 方式
- OAuth2個人連携

### 送信単位
- 月単位
- 日単位

### 差分判定
- アプリ側で更新日時から判定

### ステータス
- 未送信
- 送信済
- 送信失敗
- 再送待ち

---

## 7. ログ

### 変更履歴
- 誰が
- いつ
- 何を変更したか

### 操作ログ
- ログイン
- ログアウト
- 連携実行
- 締め処理

保存期間：3年

---

## 8. 非機能要件

- PC / スマホ対応
- Chrome / Edge / Safari
- 同時利用30名程度
- TLS必須
- 日次DBスナップショット
